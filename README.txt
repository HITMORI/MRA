Multireference alignment and bispectrum inversion code for the paper:

"Bispectrum Inversion with Application to Multireference Alignment",

by:

Tamir Bendory, Nicolas Boumal, Chao Ma, Zhizhen Zhao and Amit Singer.

First version of the paper posted to ArXiv on May 1st, 2017.

Paper: https://arxiv.org/abs/1705.00641

Code:  https://github.com/NicolasBoumal/MRA

Run main.m for a quick example.

Run experiment_***.m to generate figures from the paper.

Requirements:

 # Manopt 3.0 or later, available at http://www.manopt.org (necessary)

 # CVX 2.1 or later, available at http://cvxr.com/cvx/ (optional)
